# College Monitoring System

## Overview
A web-based College Monitoring System developed using Flask, HTML, CSS, JavaScript, and SQLite.

## Features
- Student Management
- Attendance Tracking
- Performance Analysis
- Dashboard
- Prediction Module

## Technologies Used
- Python (Flask)
- HTML
- CSS
- JavaScript
- SQLite

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the application:
   python app.py

3. Open the application in your browser.
